# Repositorio de Proyectos DAM

Este repositorio contiene una colección de proyectos y ejercicios realizados durante mis estudios en el ciclo formativo de **Desarrollo de Aplicaciones Multiplataforma (DAM)**. Está organizado por temas y tecnologías para facilitar el aprendizaje y la revisión.

## Descripción

Como estudiante de DAM, este espacio sirve para documentar mi progreso en el desarrollo de aplicaciones multiplataforma. Incluye proyectos en Java, Node.js, Spring Boot, React, Android (Kotlin), bases de datos, y más. Cada carpeta representa un módulo o tema específico del currículo.

## Estructura del Repositorio

### Acceso a Datos
- **AccesoAdatos/**: Ejercicios de acceso a datos en Java, incluyendo listas de nombres, correos y un generador de nombres con Maven.
- **Tema2/**: Proyectos de DAO (Data Access Object) para reservas, conversión de Excel a base de datos, y gestión de camisetas.

### Desarrollo Web
- **nodejs/**: Aplicaciones Node.js, incluyendo DAO y gestión de camisetas.
- **javascript/**: Ejercicios y aplicaciones web en JavaScript, como gestión de tienda, páginas web y exámenes.
- **react/**: Proyectos con React, incluyendo conceptos básicos, estado y ejercicios prácticos.
- **spring/**: Aplicaciones Spring Boot, como web apps y gestión de camisetas.

### Desarrollo Móvil y Multimedia
- **MULTIMEDIA/**: Proyectos Android en Kotlin, incluyendo actividades, llamadas SOS, dados, drawer, intents, fragments, login, navegación, progress bars, spinners, y más.

### Otros Temas
- **Tema4/**: Aplicación Flask con MongoDB y Praetorian para reservas.
- **Interfaces/**: Ejercicios de interfaces.
- **ingles/**: Contenido relacionado con inglés.
- **postman/**: Colecciones para testing de APIs.
- **RaspBerry/**: Proyectos para Raspberry Pi, incluyendo un juego de Pong en Python.
- **PSP/**: Programación de Servicios y Procesos.
- **PROYECTO/**: Proyectos finales o integradores.
- **SERVICIOS/**: Servicios y dumps de base de datos.
- **recuperacion/**: Material de recuperación.

## Tecnologías Utilizadas

- **Lenguajes**: Java, JavaScript, Python, Kotlin, SQL.
- **Frameworks**: Spring Boot, React, Flask, Node.js.
- **Bases de Datos**: MongoDB, MySQL/PostgreSQL, Excel.
- **Herramientas**: Maven, Gradle, Postman, Android Studio.
- **Plataformas**: Web, Android, Raspberry Pi.

## Cómo Usar

1. Clona el repositorio: `git clone <url-del-repositorio>`
2. Navega a la carpeta deseada.
3. Para proyectos Java/Maven: Ejecuta `mvn clean install` y luego `mvn exec:java`.
4. Para Node.js: `npm install` y `npm start`.
5. Para Android: Abre en Android Studio y ejecuta en emulador/dispositivo.
6. Para Python: `python <archivo.py>`.

Cada proyecto incluye instrucciones específicas en sus carpetas si es necesario.
